# BigModel Web Search

BigModel 联网搜索 skill（单一完整接口版本）。

## Features

- 调用 POST /paas/v4/web_search
- 默认 search_engine 为 search_pro_quark
- 支持 data.api_key 或 secret 传入 API Key
- 返回可直接用于 citation 的来源列表
