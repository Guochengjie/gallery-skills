# Python via Pyodide

A JavaScript skill that runs lightweight Python snippets through Pyodide.

## Notes

- Runtime is loaded from jsDelivr CDN.
- The Python snippet should set a `result` variable.
- Optional `variables` object is injected into Python globals.
