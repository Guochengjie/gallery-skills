# BigModel Reader

BigModel 网页抓取与解析 skill。

## Features

- 调用 POST /paas/v4/reader
- 支持 data.api_key 或 secret 传入 API Key
- 支持 timeout/no_cache/return_format 等参数
- 返回正文、标题、描述和原始 URL，便于引用
