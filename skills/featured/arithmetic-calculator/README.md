# Arithmetic Calculator

A simple JavaScript skill to evaluate arithmetic expressions accurately.

## Supported syntax

- Numbers: integers and decimals
- Operators: +, -, *, /
- Parentheses: ( )
- Unary minus: e.g. -5 + 2, 3 * (-4)

## Example payload

```json
{
  "expression": "(5 * 10) / 2"
}
```
